### 프로젝트

>  자유롭고 편리한 소통의 시작, **LIBERTYFORM**
>
>  - 'Google Form, Type Form을 벤치마킹하여
>  - 편의성, 유연성, 직관성을 모두 제공하는 설문관리 웹페이지를 만들기 위해 시작했습니다.

<br>

1. 설문관리 기본 기능은 물론
2. 설문별 **`이력관리, 설문발송, `** 기능
3. 설문 욕구를 자극하는 세련된 UI까지!
