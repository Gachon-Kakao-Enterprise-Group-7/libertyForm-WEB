### 프로젝트

>  Github과 가장 긴밀하게 연결되는 새로운 솔루션, **LOCKER**
>
>  - '개발자가 만든 개발자 블로그'를 모토로
>  - 개발자로서 가장 불편한 지점에 대한 문제의식에서 시작했습니다
>  - 개발자, 일단 넌 LOCKER에 다 때려 넣어봐. 정리는 우리가 할게 :)

<br>

1. 블로그의 기본 기능은 물론
2. 깃헙의 **`레포지토리 별로 포스트를 관리하는`** 기능
3. 포스팅 욕구를 자극하는 세련된 UI까지!
